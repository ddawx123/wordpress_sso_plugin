# wordpress_sso_plugin
WordPress SSO Plugin By JSONP - WordPress 跨域单点登录插件

## 支持特性
* [x] 跨域用户Session/Cookie共享
* [x] 跨域同步登陆/登出
* [x] 基础的CSRF/XSS防护（配合服务端）
* [x] 来路URL转发处理
* [ ] 开箱即用OOBE化处理

## 安装方法
在WordPress后台导入ZIP压缩包即可，或手动解压后上传主体插件目录到wp-contents/plugins目录。
